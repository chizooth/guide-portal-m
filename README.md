# guide-portal-m

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面集合的托管与维护入口。

## 项目简介

guide-portal-m 旨在整理和存放一批独立的 HTML 页面。这些页面可能包含信息展示、功能工具或静态内容，彼此之间没有强关联，各自独立运行。仓库本身不提供索引或导航页面，所有内容以文件形式直接归档。

## 目录说明

```
guide-portal-m/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 开源许可证
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名即页面标识。
- `assets/`：按需存放页面所需资源，建议按页面或类型组织子目录。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间无链接跳转要求，各自独立。
- 归档时请确保页面内无外部域名硬编码（如需要可改为相对路径或占位符）。
- 页面文件命名建议使用英文小写、连字符分隔，例如 `example-page.html`。

## 维护说明

- 本仓库由维护者定期更新，新增或修改页面时请提交 Pull Request。
- 提交前请确认页面在本地可正常打开，资源引用路径正确。
- 不建议在页面中加入跟踪脚本、广告代码或任何形式的用户数据收集功能。
- 如有页面不再需要，请在 `pages/` 中删除对应文件并说明原因。
